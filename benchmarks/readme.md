## 1 Results for Siemens programs

For each subject programs, the summary of results from ddmin and DDMT are provided.
In each file, the reduction result is summarized via:

* the size of input;
* the number of queries;
* the time cost;
* the resulting input.

For example, "data:3;28;0.046361207962;=>I": the size is 3, the number of queries is 28, and the time used is 0.046361207962 seconds, and the resulting input is "==>I".
